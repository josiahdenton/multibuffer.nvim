# multibuffer.nvim

This plugin experiments with multibuffers. A tab of floats are
set to the buffers corresponding to a given action.


**Currently supported actions**
- lsp diagnostics 
- symbol usage

> [!caution]
> plugin still in a "draft" state and is designed for my needs.
